# MUE Learning Categories

> Master reference for the 7 subject-material categories used across the MUE curriculum.

```
🤖 AI & Copilot     ⚡ Codex Productivity     🏗️ Pyramid Platform
📊 BI Judgment      🔗 Data & Lineage         📦 Delivery & Handoff
🧠 Retention & Readiness
```

---

## 🤖 AI & Copilot
| Aspect | Detail |
|--------|--------|
| **Focus** | Chat modes (Ask/Edit/Agent/Plan), prompt engineering, custom instructions, context management, tool use |
| **Key Skills** | Mode selection, prompt structure, instruction files, prompt files, context windows, drift avoidance |
| **Artifacts** | Prompt templates, revision logs, mode comparison notes |
| **Weeks Active** | 1–4 |

## ⚡ Codex Productivity
| Aspect | Detail |
|--------|--------|
| **Focus** | Workflow orientation, handoff fluency, context synthesis, targeted AI use, bounded automation, Codex loop mastery |
| **Key Skills** | Codex Loop (Pull → Summarize → Identify → Execute → Record), 5-fact handoff extraction, 3-question context pull, bounded prompt design, manual-vs-Codex evaluation, handoff quality rubric, prompt pattern library |
| **Exercises** | EX1: Handoff Reading (<60s extraction), EX2: Context Pull (scoped prompt refinement), EX3: State Summary (3-sentence mastery), EX4: Handoff Creation (rubric-graded), EX5: Manual-vs-Codex Comparison, EX6: Bounded Codex Simulation |
| **Prompt Patterns** | P1: Context Pull, P2: State Summary, P3: Handoff Draft, P4: Validation Check, P5: Manual-vs-Codex, P6: Change Impact, P7: Bounded Prompt |
| **Artifacts** | Handoff drafts, workflow summaries, context snapshots, Codex comparison notes, bounded prompt logs, prompt pattern templates, exercise results |
| **Weeks Active** | 1–4 |
| **Mastery Signal** | Can complete Codex Loop unassisted in <5 min; handoff passes Quality Rubric; bounds are self-enforced without reminders |

## 🏗️ Pyramid Platform
| Aspect | Detail |
|--------|--------|
| **Focus** | Model logic, deployment sequencing, QC, security, artifact movement, reviewer access |
| **Key Skills** | Model lineage, deployment preflight, artifact migration, QC execution, access policy, reviewer path |
| **Artifacts** | Deployment checklists, preflight logs, access notes, QC evidence packs, migration records |
| **Weeks Active** | 2–4 |

## 📊 BI Judgment
| Aspect | Detail |
|--------|--------|
| **Focus** | Business reasoning, metric definition, grain, validation, trusted delivery, tool-agnostic thinking |
| **Key Skills** | Business question articulation, metric selection, grain/filter/exclusion definition, output type selection, validation evidence |
| **Artifacts** | Metric lineage walkthroughs, validation records, business question notes, grain definitions |
| **Weeks Active** | 1–4 |

## 🔗 Data & Lineage
| Aspect | Detail |
|--------|--------|
| **Focus** | Source-to-model lineage, aggregation, rollups, snapshots, dependency mapping, row ownership |
| **Key Skills** | Dependency tracing, row ownership identification, deduplication logic, aggregation boundaries, double-counting risk |
| **Artifacts** | Dependency maps, row ownership notes, aggregation boundary notes, lineage diagrams |
| **Weeks Active** | 1–3 |

## 📦 Delivery & Handoff
| Aspect | Detail |
|--------|--------|
| **Focus** | Change isolation, review packages, handoff drafting, reusable assets, reviewer readiness |
| **Key Skills** | Change-slice scoping, review package preparation, handoff note generation, asset creation, reviewer path setup |
| **Artifacts** | Change charters, review packages, handoff notes, reusable assets (prompts, checklists, templates) |
| **Weeks Active** | 3–4 |

## 🧠 Retention & Readiness
| Aspect | Detail |
|--------|--------|
| **Focus** | Weekly retention checks, scorecards, proof tasks, Codex Gate, readiness classification |
| **Key Skills** | Cross-track recall, self-assessment, proof task completion tracking, gate readiness evaluation |
| **Artifacts** | Weekly scorecards, retention reviews, readiness reports, Codex Gate checklists |
| **Weeks Active** | 1–4 (cumulative) |
